<a name="english"></a>
## English

### Overview
CodeQuest Dojo is an interactive JavaScript learning game designed to help beginners learn programming through fun challenges and adventures.

### Features
- **Interactive Coding Challenges**: Learn JavaScript by solving real coding problems
- **Level Progression**: Progress through stages and level up your coding skills
- **Robot Companions**: Collect different robot characters as you advance
- **Sound Effects**: Immersive audio feedback for actions and achievements
- **Beautiful UI**: Aurora Borealis themed interface with modern design

### How to Play
1. Open `index.html` in your web browser
2. Follow the JavaScript challenges presented
3. Write code to solve each puzzle
4. Level up and unlock new stages

### Files
| File | Description |
|------|-------------|
| `index.html` | Main game application |
| `codequest-reference.html` | JavaScript reference guide |
| `robot_*.png` | Robot character images |
| `*.mp3` | Sound effects and music |

### Technologies Used
- HTML5
- CSS3 (with custom Aurora Borealis theme)
- JavaScript
- Supabase (for backend services)

## 日本語

### 概要
CodeQuest 道場は、初心者がプログラミングを楽しいチャレンジと冒険を通じて学べるインタラクティブなJavaScript学習ゲームです。

### 特徴
- **インタラクティブなコーディングチャレンジ**: 実際のコーディング問題を解きながらJavaScriptを学習
- **レベル進行システム**: ステージをクリアしてコーディングスキルをレベルアップ
- **ロボットコンパニオン**: 進むにつれて様々なロボットキャラクターを収集
- **サウンドエフェクト**: アクションや達成時の臨場感あるオーディオフィードバック
- **美しいUI**: オーロラをテーマにしたモダンなデザイン

### 遊び方
1. ウェブブラウザで `index.html` を開く
2. 表示されるJavaScriptチャレンジに挑戦
3. パズルを解くコードを書く
4. レベルアップして新しいステージを解放

### ファイル構成
| ファイル | 説明 |
|----------|------|
| `index.html` | メインゲームアプリケーション |
| `codequest-reference.html` | JavaScriptリファレンスガイド |
| `robot_*.png` | ロボットキャラクター画像 |
| `*.mp3` | 効果音と音楽 |

### 使用技術
- HTML5
- CSS3（カスタムオーロラテーマ付き）
- JavaScript
- Supabase（バックエンドサービス用）
